# Cricket 300 Draft

A browser-based cricket drafting and simulation game.

Draft an XI from randomized country and decade rolls, then simulate whether the side can score more than 300 runs in 20 overs. The game includes:

- Easy, Medium, and Hard modes
- Era-adjusted player ratings
- A large multi-era cricketer database
- Mandatory wicketkeeper selection
- Position eligibility rules
- Ball-by-ball simulation with speed controls and skip-to-end
- Post-match analysis of what worked and what could improve

## Play

Open `index.html` through GitHub Pages.
